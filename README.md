# Mixed Bevs Reviews - Learn, create, and rate mixed drinks

## Frontend:
- NextJS
- Tailwind CSS
- React
- TypeScript

## Backend
- Supabase

# Features
- Users can create and leave a rating on mixed drinks
- Discover and learn how make mixed drinks
- Get a random mixed bev daily
